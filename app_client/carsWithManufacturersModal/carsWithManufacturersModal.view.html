Cars With Manufacturers Application

Open Project website

Git Hub Link: https://github.com/chaitanyakrishnabojja/MEAN-Cars-Api-App

• Developed an Angular application using AngularJS, this application allows the users to get the list of car manufacturers for a given year, I used the cars API to get the list. when the user enters the year in the header part of the application and “Get Manufacturers!” button is pressed, list of manufactures will be displayed along with Make Name, Make ID, Manufacturer ID, Manufacturer Name

• Users can also filter the Make Name’s using the Filter Results. Users can also filter the manufacturers list by using check boxes on the left

• When any of the Make Name is clicked from the list, all the models for that Make Name will be displayed on the right hand side of the webpage

• The total, available and booked rooms count will change based on the user bookings• We can also filter the model names using the right side filter

• The filtered count will be displayed at the top

• The check boxes count will change based on the year selected in header part

• Used Angular Services to get the JSON data from the cars API
