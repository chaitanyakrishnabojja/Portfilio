MoCell Application

Open Project website

Git Hub Link: Git Hub

• MoCell data-driven web application using Node.js, Express, AngularJS and MongoDB

• It is a fully functioning Angular web application in which I’ve used all layers of the MEAN stack to create MoCell

• I have also included Bootstrap to create a responsive layout. MoCell will list all the products in categories (mobile, laptops and accessories), where people can use all the CRUD (create, read, update, and delete) for a single category

• It will also display Product name, Price, description, and a rating for each product

• Users will be able to submit ratings and reviews

• I designed and developed API’s to make service requests to my MongoDB Database

• I used Single Page Application (SPA) to develop this application
