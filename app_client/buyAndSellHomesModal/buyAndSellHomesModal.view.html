Buy And Sell Homes Application

Git Hub Link: Git Hub

• Developed an app that will display all of the houses in an area

• The app will contain a home screen, an activity with a list of houses listed as per the ascending order of distance(i.e., the nearest house will be displayed first in the list) and a fragment with details about each house

• There will be a House class that contains member variables like price, description, etc

• The detail activity will create an instance of a House and populate the variables using data from an SQLite database

• The app includes the Google maps API to show the location of the house on the map inside a Fragment

• The users can take the picture of the house and can put it in the app for sale

• The uses can also buy the house they like and once the house was bought, the house goes to the bought homes list

• This app also allows the users to contact the app maintenance team
