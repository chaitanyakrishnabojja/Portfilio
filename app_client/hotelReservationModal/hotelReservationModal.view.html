Hotel Rooms Booking Project

Open Project website

Git Hub Link: https://github.com/chaitanyakrishnabojja/JavaScript-projects/tree/master/JavaScriptHotelBookingProject

• Developed this application using javaScript and jQuery

• This application consists of booking page with 3 types of rooms(Silver, Gold, Platinum) with total, available and booked rooms count

• This application allows the users to book the type of room they want

• The total, available and booked rooms count will change based on the user bookings

• Also, added validations for every input fields in the reservation form using JavaScript
